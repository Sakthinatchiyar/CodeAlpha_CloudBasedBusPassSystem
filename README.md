# Cloud-Based Bus Pass System

## Project Overview

The Cloud-Based Bus Pass System is a web application developed using Flask and SQLite that allows users to book bus passes online. The system generates a unique digital pass for every booking, preventing ticket loss, theft, and incorrect pricing. The application is designed to be hosted on the cloud, providing a scalable and reliable ticket booking experience.

## Features

* Online bus pass booking
* Automatic ticket price calculation
* Unique Pass ID generation
* Digital pass management
* Secure database storage
* Prevention of duplicate ticket records
* Cloud deployment support
* User-friendly interface
* Fast and reliable booking process

## How It Works

1. User opens the application.
2. User clicks on "Book Pass".
3. Passenger details are entered.
4. System calculates ticket price automatically.
5. A unique Pass ID is generated.
6. Booking details are stored in the database.
7. Digital pass is displayed to the user.
8. Pass information can be retrieved whenever required.
